# Machine Learning Practitioner
This repository contains my solutions for the challenges from DIO course Machine Learning Practitioner course.

Each folder name starts with the sequential number following the course order.

- Challenge 1 - Apply the Transfer Learning method in a Deep Learning network in the Python language. 
- Challenge 2 - Apply dimensionality reduction to one image. 
- Challenge 3 - Calculate evaluation metrics for classification model using confusion matrix.
- Challenge 4 - Yolo Detection
